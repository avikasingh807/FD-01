# 🌐 Landing Page Project

This is a responsive and visually appealing landing page built using HTML and CSS.

## 🚀 Features

* Clean and modern user interface
* Responsive design (mobile, tablet, desktop)
* Navigation bar (Home, About, Contact)
* Hero section with call-to-action button
* Features section with icons and descriptions
* About and Contact sections
* Images used for better visual appearance

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📁 Project Structure

Landing-Page/
│── index.html
│── style.css

## 🎯 Purpose

The main objective of this project is to practice front-end web development by creating a simple and responsive landing page.

## 📱 Responsive Design

This website is fully responsive and works smoothly on different screen sizes including mobile devices.
