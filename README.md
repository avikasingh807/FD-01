# 🌸 GlowCare Landing Page

GlowCare is a modern and responsive skincare landing page designed to showcase organic and dermatologist-tested skincare products. This project focuses on clean UI, attractive design, and user-friendly layout.

---

## ✨ Features

* 🌿 100% Organic Skincare Section
* 🧴 Dermatologist Tested Products
* 🚚 Fast Delivery Information
* 💻 Responsive Design (Mobile + Desktop)
* 🎨 Modern UI with soft skincare theme
* 🧭 Smooth Navigation (Home, About, Contact)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts (Poppins)
* Pexels Images

---

## 📂 Project Structure

GlowCare/
│── index.html
│── style.css
│── README.md

---

## 🚀 How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser
